This project implements a CHIP-8 based virtual gaming console on an STM32 microcontroller using bare-metal programming. It demonstrates low-level peripheral interfacing including SPI communication for display and SD card integration for ROM storage. The system emulates key architectural components such as registers, memory, and instruction cycles to execute classic games. Designed with a hardware–software co-design approach, it highlights real-time embedded system development and efficient resource utilization. The project showcases practical application of embedded systems, firmware design, and graphics rendering on constrained hardware.
